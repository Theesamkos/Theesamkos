### 🏆 Featured Projects

#### 🤖 [Nerdy Autonomous Ad Engine](https://github.com/Theesamkos/nerdy-autonomous-ad-engine)
An autonomous pipeline that generates, evaluates, self-heals, and progressively improves Facebook & Instagram ad copy without human intervention. Built for the Nerdy / Varsity Tutors Gauntlet challenge.
- **Tech:** TypeScript, Node.js, LLMs
- **Highlights:** Self-healing feedback loops, quality ratchet, performance-per-token tracking, and agentic orchestration.

#### 🤝 [CollabBoard](https://github.com/Theesamkos/Collab-board) & [CollabBoard Backend](https://github.com/Theesamkos/collab-board-backend)
A real-time collaborative whiteboard application featuring a powerful AI agent to interpret natural language commands and manipulate the whiteboard state.
- **Tech:** TypeScript, React, Python, FastAPI, LangChain, Claude 3.5 Sonnet
- **Highlights:** Object creation/manipulation, layout arrangement, template generation, and board summarization via AI.

#### 🏒 [Coaching Assistant](https://github.com/Theesamkos/Coaching-Assistant-)
An AI assistant for hockey coaches to manage schedules, track player performance, and streamline training operations.
- **Tech:** React 18, TypeScript, Vite, Tailwind CSS, Zustand, Firebase
- **Highlights:** Comprehensive workflow management, drill assignments, and progress tracking.

#### 🪙 [Claude Code Token Optimizer Skill](https://github.com/Theesamkos/claude-code-token-optimizer-skill)
A production-ready Python package and command-line tool to optimize Claude API token usage when generating code, saving 50-80% on token costs.
- **Tech:** Python, Claude API
- **Highlights:** Intelligent model routing, pre-execution cost estimation, and automatic quality validation.
